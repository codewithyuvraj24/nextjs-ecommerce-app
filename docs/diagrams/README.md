# Architecture Diagrams

This directory contains **draw.io** (`.drawio`) diagrams that document the architecture of the e-commerce application.

## Diagrams

| File | Description |
|------|-------------|
| [`system-architecture.drawio`](./system-architecture.drawio) | High-level system overview — frontend, backend, database, and external services (Razorpay, Shiprocket) |
| [`database-er.drawio`](./database-er.drawio) | Entity-Relationship diagram for all 10 PostgreSQL tables with columns, keys, and relationships |
| [`frontend-components.drawio`](./frontend-components.drawio) | Component tree showing layout hierarchy, pages, shared components, and Zustand stores |
| [`api-routes.drawio`](./api-routes.drawio) | Backend request lifecycle — middleware pipeline, route groups, controllers, and models |

## How to Open / Edit

1. **Online** — Go to [app.diagrams.net](https://app.diagrams.net) → *File → Open from → Device* → select a `.drawio` file
2. **Desktop** — Install [draw.io Desktop](https://github.com/jgraph/drawio-desktop/releases) and double-click any `.drawio` file
3. **VS Code** — Install the [Draw.io Integration](https://marketplace.visualstudio.com/items?itemName=hediet.vscode-drawio) extension to edit diagrams directly in your editor
